# Space Simulator

Early development space simulation project made in Godot.  

## Current Features
- LOD quadsphere generation  
- Patch-based terrain  
- Colour map support  
- Height map support  
- Basic lighting and normals  
- Simple debug tools (wireframe, patch view)

## Planned Features
- Improved terrain accuracy  
- Atmosphere rendering  
- Scatter objects  
- Rocket parts and basic flight  
- Simple orbital physics  
- UI and general tools  
- More optimized LOD updates

## Project Status
Still in foundational stages.  
Main goal right now is getting planets stable and performant.
